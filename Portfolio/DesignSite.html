Colors:

Red: #FF5733

Used as visual representation of an active link. Used to draw attention.

Green: #33FF57

Just a fun color to have that can add a nature-esque look to a page!

Blue: #3357FF

This color shall mainly be used to identify clickable links on the page itself.

Yellow: #FFFF33

Yellow is a fun color to have, but it can be hard to read on some backgrounds.

Orange: #FFA500

Orange, much like yellow, should be used only to liven things up! Look into using background color attributes when using tough to read colors!
